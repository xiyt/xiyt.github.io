IntelliJ IDEA 开发助手插件DevHelper发布啦，充分提高工作效率

DevHelper是一个能够提高效率的IntelliJ IDEA开发助手插件，使用该插件把开发过程中常用的开发工具直接集成到IDE中，使用起来既快捷又方便，能够大大提高工作效率。目前提供的功能如下，博主会及时更新和添加新的功能，敬请期待

1、生产密码

可以指定大小写字母、数字、特殊符号任意组合，长度任意指定，点击按钮一键批量生成密码；

2、生成序列

开发过程或者数据库操作过程中，经常需要填入序列，该功能可以根据起始值、步长、位数生成一系列序号，并可添加前后缀；

3、Base64编码和解码

Base64是最常用的编码方式之一，在开发过程中一定会用到，该功能可以对内容进行Base64编码和解码，并可以指定编码格式，以及URL安全选项；

 

4、散列哈希加密

该功能支持MD5、SHA-1、SHA-224、SHA-256、SHA-384、SHA-512、HmacMD5等常见的散列哈希算法，可以选择不同的算法对内容进行不可逆加密

5、文本处理

文本处理是开发过程中最常用到的功能，该功能提高了大量常用的文本操作，可以对行文本进行格式化(去除空白、空行、重复行)、行编辑(合并、拆分、命名转换等)、行排序(按照字母或数据升序降序)等。当然如果你正则表达式和文本编辑器本身用的很熟练的话，也可以自己完成这些操作。但是在这里效率也许会更高。

6、生产UUID

在数据库手工造数据、生产随机字符串亦或者是生产某个Key和Token的时候，经常会用到UUID，该功能批量生成大写/小写 UUID，并可以选择是否带有分隔符

7、格式化

集成了常用的文本格式化功能，目前支持Json、xml、html的格式化

8、RSA非对称加解密

RSA非对称加解密在程序开发过程中也是经常用到的，但是在开发过程中，最重要的并不是如果实现RSA加解密，而是如何在多个终端不同语言（安卓、IOS、Java、Javascript等）下RSA算法统一。该功能提供了相对标准的是RSA加解密算法，并且和该网站能够互相加解密。目前该功能支持RSA的私钥/公钥分别进行加密或者解密，并且可以生成私钥和公钥对

1、打开IntelliJ IDEA中：File->Settings->Plugins，然后搜索DevHelper即可安装。由于官网发布需要审核，所以版本会延时，可直接采用第二种方式，直接下载最新版安装。

2、下载最新格式安装包，打开File->Settings->Plugins，然后选择 Install Plugin from Disk… ，然后选择下载好的安装包，即可完成安装。

当前最新版本：DevHelper-1.0.1

该插件已发布到IntelliJ插件中心，插件首页为：https://plugins.jetbrains.com/plugin/15322-devhelper
