数据库管理工具 JetBrains DataGrip 2018.2.3 特别版下载

简单介绍下DataGrip：

DataGrip出自JetBrains公司，是一款跨平台的数据库管理客户端工具，可在Windows，OS X 和 Linux上使用；同时支持多种数据库：SQL Server，Oracle，PostgreSQL，MySQL，DB2，Sybase，SQLite，Derby，HyperSQL和H2；方便连接到数据库服务器，执行sql、创建表、创建索引以及导出数据等。类似的IDE很多，比如Navicat、dbeaver，DataGrip功能上更为强大一些。何出此言，因为出自同一个公司的 Intellij IDEA 是最好的 java 开发 IDE，如果之前有使用 IDEA，则一定会很快上手使用 data grip。

 

现在运行压缩包中的主安装文件，安装完成后的破解方法如下：
