windows中创建Redis集群方法(附Redis集群下载)

虽然在windows中安装Redis的机会很少，但是当服务器或者测试环境Redis集群down掉的时候，作为备用开发方案，还是很有必要的。所以这里记录一下我在windows中安装Redis集群的过程和步骤，并附上配置好的集群的下载链接。

  • 下载windows版本的Redis，地址：https://github.com/MicrosoftArchive/redis/releases
  • 安装Ruby
  • 下载Ruby Reids 驱动文件：redis-3.2.2.gem，地址：https://rubygems.org/gems/redis/versions/3.2.0
  • 使用Ruby的gem命令安装redis-3.2.2.gem驱动：
  • 下载搭建Redis集群说是用的脚本redis-trib.rb，地址：https://github.com/MicrosoftArchive/redis/blob/3.2/src/redis-trib.rb
  • 创建Redis集群节点的配置文件，以其中一个为例：
  • 注意：如果集群想设定密码，方法在这里：https://www.cnblogs.com/linjiqin/p/7462822.html，因为我是本机测试使用，直接在 client.rb 文件里的修改了密码，这样就不需要再集群创建完成后在一个一个节点的设定密码了。
  • 提供一个一键启动集群的命令行
