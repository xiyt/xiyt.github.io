IntelliJ Idea 2019.1.3永久激活方式，JAR文件分享

使用方法: 1、先下载压缩包解压后得到jetbrains-agent.jar，把它放到你认为合适的文件夹内。

2、修改IntelliJ Idea安装目录bin文件夹里的idea64.exe.vmoptions文件，如果是32位系统，就对应的是idea.exe.vmoptions文件。在该文件末端添加一行：

-javaagent:E:/Program Files/ideaIU/jetbrains-agent.jar

注意路径要根据实际情况填写。另外如果你之前已经添加过-javaagent这个参数了，就把后面的jar路径改为最新的就行了。保证该文件中只能有一个-javaagent

3、启动IntelliJ Idea，打开注册窗口： 支持两种注册方式：License server 和 Activation code: 1). 选择License server方式，地址填入：http://jetbrains-license-server （应该会自动填上） 或者点击按钮：”Discover Server”来自动填充地址。 2). 选择Activation code方式离线激活，请使用：ACTIVATION_CODE.txt 内的注册码激活 如果激活窗口一直弹出（error 1653219），请去hosts文件里移除jetbrains相关的项目

这个最新的jar包在Idea 2019.1.3版本测试通过。

jetbrains-agent-v2.1.1-支持idea版本为-2019.1.3
