[下载]iPhone中Surge上网应用PP正版下载+使用教程

至于Surge是什么，大家自行百度吧。我已经安装成功，并可以正常使用了。由于各种原因该应用已经从App Store下架了，而且PP助手里也找不到了，这个也是我在网络上搜集到的，亲测可用，提供下载地址和使用方法给大家。另外本站提供Surge的配置文件和SSEncrypt.module的下载和使用：

http://www.daxiblog.com/main-normal.conf

http://www.daxiblog.com/SSEncrypt.module

声明：conf配置文件和SS mosule最早来自作者的网站，现在被作者删了，我也是在abclite.cn上找到后，放在服务器上做分流使用。安全性请自己评估后再决定是否使用。

Surge下载地址：

本地下载：Surge@大席的博客

Dropbox下载：Surge@大席的博客

使用说明：

1、iPhone不需要越狱，通过电脑版PP助手安装Surge - Web Developer Tool and Proxy Utility-1.0.1(PP正版).ipa文件 2、在电脑端记得点击PP助手右上角的修复闪退功能 3、在iPhone端打开Surge点击编辑，选择 4、输入：http://www.daxiblog.com/main-normal.conf 5、导入成功之后我们会看到出现了一个叫main-normal的配置文件,点击该文件会提示Error,然后我们选择,之后找到项,按照如下格式填： Proxy = custom,1.2.3.4(请替换为你自己的服务器ip),443(替换你自己的端口号),method(加密方式，比如aes-256-cfb),password(密码),http://www.daxiblog.com/SSEncrypt.module 6、 这样基本就配置好了,只需要点击Start,然后允许添加V-P-N就可以科学上网了,状态栏上会有v-p-n标识

※对于proxy-server-01文件的说明： 如果你有多个代理服务器，你就可以创建多个这样的文件，把服务器信息填写进去，至于规则都是引用main-normal.conf里的规则配置的。 这样就实现了配置和服务器分开了。
