我的PC端和移动端的协同工作方式，开展移动办公

        今天我想和大家分享的是我的PC端和手机端的协同工作方式，如今智能手机的配置越来越强大，可以运行的手机软件也越来越多，除了游戏、音乐、购物等应用外，也包括各种办公软件，如微软Office、邮件客户端、文档管理工具等，再加上现在大家是手不离机，随身随地都会带着手机，所以必要时把手机作为我们的第二工作地，对提高我们的工作效率和实施数字化工作都是有帮助的。

目录：

  • 邮件
  • 待办清单/记事贴
  • 知识管理
  • 文件同步与共享
  • 临时文件传输
  • 结束语：充分利用移动终端，开展移动办公

一、邮件的协同工作

  • 桌面端：Foxmail邮件客户端
  • 移动端：GMAIL手机邮件客户端
  • 协作方式：
    1. 邮件转发，在Foxmail中配置邮件自动转发到我的GMAIL邮箱
    2. 紧急邮件，手机端直接回复
    3. 会议中需要查看的资料类邮件，直接在手机端打开查看，就不需要抱着笔记本电脑去会议室了

延伸阅读： 我的邮件整理方法

二、待办清单/记事贴的协同办公

  • 桌面端：Google keep网页版
  • 移动端：Google keep手机版
  • 协作方式：
    1. 在网页版中添加待办事项、提醒、稍后阅读的网址
    2. 桌面端Chrome中可以收到提醒通知
    3. 手机端的Google Now同样能收到提醒通知
    4. 在手机端打开稍后阅读的网址，进行阅读

延伸阅读： 我的Google Keep七大功能使用技巧

三、知识管理的协同办公

  • 桌面端：印象笔记PC版
  • 移动端：印象笔记移动版
  • 协作方式：
    1. 桌面端添加笔记、图片、文档等知识信息
    2. Chrome插件剪切网页信息、网页图片
    3. 手机端浏览器中，使用分享功能剪辑网页信息到手机版印象笔记
    4. 手机端使用离线笔记功能，随时随地查看重要笔记

四、文件同步共享的协同办公

  • 桌面端：Dropbox桌面版
  • 移动端：Dropbox移动版
  • 协作方式：
    1. PC端Dropbox作为同步盘，存放最近工作的相关文档；
    2. 手机端使用Office软件、PDF查看软件，查看编辑Dropbox同步的工作文档
    3. 网页版Dropbox中，查看文档的修订版本

延伸阅读： U盘/网盘/移动硬盘中文件、资源的整理心得

五、临时文件传输的协同办公

由于平时工作中，一直使用QQ作为即时通讯工具，所以这里使用它作为我的临时文件传输工具，如果你使用其他的通讯工具，也可以查看一下是否有这样的功能，就不需要再去使用第三方的文件传输工具了。

  • 桌面端：腾讯QQ桌面版
  • 移动端：QQ手机版
  • 协作方式：
    1. 桌面端QQ中提供的功能，能够方便的将电脑中的图片、文件、截图、文本、剪切本里的信息，直接发布到手机QQ中
    2. 打开手机QQ就可以收到来自桌面端的文件等信息了；
    3. 同样在手机QQ中，也可以很方便的把手机中的图片，文件发送到电脑端。例如：用手机拍了一个头像照片，然后直接通过手机QQ发送到电脑端，经过裁剪处理后，就可以作为网站头像了。

结束语：

对于移动办公，PC端与移动端信息的同步是关键，我们可以借助好的软件和应用来实现，强大的移动端配置也为我们提供了硬件方面的保障，接下来就是我们自己了，提高移动办公的意识，改进我们的工作方式，提高工作效率，让自己轻松工作。
