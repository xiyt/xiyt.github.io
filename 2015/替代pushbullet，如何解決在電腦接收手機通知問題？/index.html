替代Pushbullet，如何解決在電腦接收手機通知問題？

最近我在思考著要用其他 App 來取代「 Pushbullet 」的幾個重要功能，倒也不是「 Pushbullet 」開始轉換付費模式，一般來說我都很支持軟體付費。而是因為不知道什麼原因，最近我的幾台電腦接收「 Pushbullet 」手機通知的同步都有問題，這帶給我一些小困擾。

我的需求是這樣的：在我工作時我的手機一定是關靜音，而且我習慣把大多數 App 通知都關掉，只保留最重要的通知來源（參考：如何一鍵關閉「所有」 Facebook LINE 遊戲邀請通知），這時候，就表示工作時出現在我的手機上的通知，一定是相對重要的通知，例如已經過濾過的 LINE 重要工作群組通知，或是經由 Google Inbox 過濾後的重要郵件通知。

但是我的手機又關靜音，我要如何一邊在辦公桌電腦工作，一邊不需要拿起手機，就能看到是否有重要訊息、重要電話進來，並判斷是否要第一時間處理呢？

之前的「 Pushbullet 」便能解決這個問題，而現在在 Android 手機上如果要找一個替代品，那麼可以試試看「 Desktop Notifications 」。

而「 Pushbullet 」另外一個關鍵功能：跨手機、電腦與不同平台裝置的「同步複製貼上」，則可以用另外一個 App 與服務來解決：「Pasteasy 電腦手機圖文同步剪貼！多裝置工作者神器」。

1. 「 Desktop Notifications 」安裝設定超簡單，跨平台可用

安裝設定「 Desktop Notifications 」的過程非常簡單，而且可以在 Android、 Windows、 Mac 與 Chrome OS 等平台上使用。

首先，我們在 Google Chrome 瀏覽器上安裝「 Desktop Notifications 」的套件，並登入你的 Google 帳號來執行同步。

接著在 Android 手機上安裝「 Desktop Notifications 」的 App ，按照 App 首頁的教學開啟通知存取功能，並登入同一個 Google 帳號，這樣就完成設定。

2. 即時同步手機訊息，可在電腦開啟

接下來你在 Android 手機上接收到的通知，就會傳送到你的 Windows、 Mac 等電腦上囉！

而且「 Desktop Notifications 」可以支援在電腦點擊通知卡片，立刻打開相關的雲端服務，例如接收到 Google Hangouts 的通知，可以直接在電腦開啟 Hangouts 。

當然，在這部分 Pushbullet 可以做到電腦直接回訊（不再只會推播！ Pushbullet 搭起 PC回應Android 橋樑），又更加強大，我之後應該還是會關注 Pushbullet 的發展，如果他解決我的同步問題，說不定我還是會回頭使用 Pushbullet 。

3. 設定只在 WiFi 環境同步訊息

另外一個我會在「 Desktop Notifications 」開啟的設定是限制只在 WiFi 環境才同步手機通知到電腦，因為我通常是在辦公室或家裡才需要這個功能（坐下來用電腦時），而這時候應該都是登入無線網路的狀態。

4. 限制某些 App 的訊息不同步到電腦

最後一個重要的設定，就是在「 Desktop Notifications 」的「 Apps 」頁面，可以限制某些 App 的通知不要傳送到電腦，以免造成額外的干擾，只要把這些 App 的通知同步全部關閉即可。

「 Desktop Notifications 」是一個免費、易用、有效的 Android 手機通知同步到電腦的服務，有需要的朋友，想替換 Pushbullet 的朋友，可以試試看。
