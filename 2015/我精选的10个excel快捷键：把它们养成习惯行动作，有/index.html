我精选的10个Excel快捷键：把它们养成习惯性动作，有效提高工作效率

在我们日常工作中，Excel应该是我们常用的办公软件之一了，在处理数据、统计数据、制作图表等方面有着很强的功能，今天我想分享一下我常用的Excel快捷键，其实在网络上搜索”Excel快捷键”都能找到很多相关的文章，例如”Excel快捷键大全”、”30个常用的Excel快捷键”等，都很全面的整理了Excel的各种快捷键功能，很好。但是今天这篇文章我想有所不同，其实我们在日常使用中，也就只有很小一部分快捷键能够熟练使用，这篇文章我精选了十组我觉得最重要、最能提高效率、  最易养成使用习惯的快捷键，分享给大家，希望能提高我们的工作效率。

Excel快捷键精选：

  • F1

我觉得这是你最应该学会也是最重要的快捷键，也就是打开Excel的帮助对话框，在这里你可以搜索关于Excel的各种问题、教程、函数使用说明等信息，而且质量很高，所以大家一定要善用自带帮助功能，不用每次都去百度、Google上搜索

  • F2

Windows中我们使用F2来重命名文件/文件夹，在Excel里可以快速的编辑当前活动单元格，并将光标放在单元格内容的结尾，也可以记忆为把单元格的内容”重命名”。

  • F4

重复上次操作，例如：

1、对一个单元格设定了颜色格式，在另外一个单元格上点击F4就会设定相同的颜色格式到该单元格上

2、想在两行之间插入一些行，我们只需要插入一行，然后按住F4不放，就会不断的继续插入多行

  • Ctrl + D 和 Ctrl + R

这一组是快速复制填充单元格的快捷键，Ctrl+D是快向下(Down)复制，Ctrl+R是快速向右(Right)复制

1、复制一个：如果选择了一行/单元格 ，则将上面一行/单元格的内容，复制到当前选择行/单元格 ；

2、复制一片：如果选择了多行/单元格 ，则将选中区域的第一行/单元格 内容复制到下面的行/单元格；

  • Alt + Enter

在单元格内换行，这个不用多说。这个我删除了，因为它不是一个提高效率的快捷键，而是一个功能性的快捷键。

  • ESC

不用总是去动你的鼠标，所使用ESC按键，这是个重要的”退出”、”取消”快捷键，在任何时候想退出正在编辑或使用的功能时，都可快速的点击一下ESC按钮，退出当前的操作

  • Ctrl + ; 和 Ctrl + Shift + ;

快速的在单元格里插入当前日期和时间

  • Ctrl + B 、Ctrl + I 和 Ctrl + U

最常用的三种文本格式化，加粗、斜体、下划线，自然有最方便的快捷键。其实这个不光在Excel里是这样的设置，在其他的编辑器、WORD、印象笔记里等里都是这样的这是设定，已经成了惯例了。

  • Ctrl +1

快速打开设置单元格格式对话框，大家肯定猜的出来，肯定还有Ctrl+2、3…，确实有，但是为什么我只推荐Ctrl+1呢，因为设置单个或一个区域的单元格是一个很常用，很重要的设置，建议大家记住并能熟练使用，其他的可以不用记了，要知道花时间和精力去记忆也是一种成本。

  • 定位：Home 和 Ctrl + Home

快速定位到当前行的第一个单元格和工作表的第一个单元格，这是我介绍的唯一一个快速定位的快捷键，因为行首和工作表的开头，是我们最经常要定位到的地方。

  • 选择：Shift + 方向键 和 Shift + Home

Shift+方向键：能够快速已当前单元格为起点，快速向四周选择区域

Shift+Home：能够已当前单元格为终点，快速选择到所在行的行头

结束语

Excel的快捷键很多，大家不用一个一个专门的记忆，找几个自己常用的，用的最多的，然后切实应用到自己的操作中，并且养成习惯性动作，就能有效的提高我们的效率。
