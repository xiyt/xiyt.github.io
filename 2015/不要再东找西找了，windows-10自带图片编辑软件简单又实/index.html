不要再东找西找了，Windows 10自带图片编辑软件简单又实用

        最近想把之前旅游的一些照片打印出来留作纪念，发现一些照片确实需要修正一下，有的曝光不足光线太暗，有的需要裁剪到合适的尺寸，有的照片有点儿倾斜，所以我准备自己动手做一些简单的处理工作。因为自己并不是专业的数码照片处理人员，所以类似PS、LR这样的软件，我根本不会用，于是我开始在网上搜寻各种轻量级的图片编辑软件，例如手机版的Snapseed、网页版的泼辣修图、国内的美图秀秀等，类似的软件很多，但是都不是很满足我的要求：

  • 轻量级的、免费的，具备基本修图功能即可；
  • 有PC版的，Snapseed确实不错，但是找不到PC版本；泼辣修图也不错，但是Windows版的是要收费的
  • 美图秀秀界面确实不喜欢

后来，在使用Windows 10查看图片时，发现是可以编辑图片的，点击编辑按钮后，就打开了Windows 10的图片编辑功能了，使用后感觉能够很好的满足我的要求，下面我把我的使用感受分享给大家：

打开Windows 10的图片编辑界面后，整个界面分为左中右三部分，分别是主功能项、图片、子功能项，顶部还有一个工具条用来保存图片以及和原始图片进行对比，主要功能包括：

1、Windows 10图片编辑的基本修图

点击左侧的基本修复，右侧就会列出来常用的图片修改选项：

  • 增强：自动修复图片
  • 旋转：调整整张图片的方向
  • 裁剪：调整图片大小
  • 拉直：用于摆正图片中的景物，缺失的部分会被自动填充
  • 红眼：去除红眼
  • 润饰：可以去掉图片中对于的人或物，经常使用

2、滤镜

Windows 10 图片编辑工具自带了六种滤镜供大家选择，相比及Snapseed和泼辣修图都少了很多，只能说是有这个功能而且，并不强大

3、光线

Windows 10图片编辑中提供了一下四中光线调整功能，都可以通过滚动数据调整数值来实现，操作起来非常方便：

  • 亮度：调整图片的整体亮度，常用于曝光不足或曝光过度的情况下，调整亮度；
  • 对比度：图片的对比度调整；
  • 突出显示：用于调整图片中明亮部分的光线的强弱；
  • 阴影：用于调整图片中昏暗部分的光线的强弱；

4、彩色

该功能用于调整图片中颜色的各种参数：包括色温、色调、饱和度、局部色彩增强，如下图所示：

5、效果

Windows 10的图片编辑中的包括两种效果功能：

一个是晕影，用户将图片四角变成暗色来是主题更加突出；

一个是选择性聚焦，将图片中的某个人或物变为焦点，模糊其余部分；
