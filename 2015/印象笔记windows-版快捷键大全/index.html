印象笔记Windows 版快捷键大全

印象笔记可以完全只用键盘操作。下面的列表中呈现了所有可供印象笔记 Windows版使用的快捷键。

全系统

你可以在使用其它应用时使用以下全系统快捷键（只要印象笔记处于运行状态）。你可以在工具 > 选项 > 全局热键选项卡中重新定义这些快捷键。

快捷键 操作
Ctrl + Alt + N 新建笔记
Win + Shift + F 在 印象笔记 中查找
Win + PrintScreen 屏幕截图
Win + A 复制所选内容
Ctrl + Alt + V 从剪贴版粘贴

应用内

以下快捷键仅在使用印象笔记应用程序时有效。

快捷键 操作
Ctrl + N 新建笔记
Ctrl + Shift + I 新建手写笔记
Ctrl + Shift + U 新建音频笔记
Ctrl + Shift + W 新建拍照笔记
Ctrl + Shift + J 新建截屏
Ctrl + Alt + C 新群聊
Ctrl + Shift + T 新建标签
Ctrl + Shift + N 新建私人笔记本
Ctrl + P 打印
Ctrl + Shift + P 打印预览
Ctrl + Alt + A 切换用户
Ctrl + Q 退出
Ctrl + Shift + Alt + W 工作群聊
Ctrl + Shift + Alt + N 笔记
Ctrl + Shift + Alt + B 笔记本
Ctrl + Shift + Alt + T 标签
Ctrl + Shift + Alt + H 企业主页（仅支持印象笔记企业版）
Ctrl + Shift + Alt + A 地图集
Ctrl + Shift + Alt + M Market
Ctrl + Shift + Alt + O 印象最前线
F10 在左侧边栏显示/隐藏
F11 显示/隐藏笔记列表
Ctrl + F5 列表视图
Ctrl + F6 摘要视图
Ctrl + F7 卡片视图
Ctrl + Shift + F5 选择在笔记列表侧边栏或底部显示笔记预览
Ctrl + F11 显示/隐藏笔记列表
Shift + Alt + N 跳转到笔记本
Shift + Alt + T 跳转到标签
Ctrl + F10 显示搜索说明
Alt + F4 关闭子窗口/隐藏主窗口（最小化到托盘）
F9 同步
F1 在线帮助
F2 重命名当前所选笔记本，笔记，标签或已保存搜索
F3 将鼠标放到活动笔记的标签中
F5 切换笔记列表视图（表格，混合模式，缩略图）

笔记编辑器

以下快捷键在你编辑一条笔记时会起效。

快捷键 操作
Ctrl + Z 撤销
Ctrl + Y 重做
Ctrl + X 剪切
Ctrl + C 复制
Ctrl + V 粘贴
Ctrl + Shift + V 粘贴为纯文本
Delete 删除
Ctrl + A 全选
F6 搜索笔记
Ctrl + Shift + A 重置搜索
Ctrl + F 在笔记中搜索
Ctrl + G 查找下一处
Ctrl + Shift + G 查找上一处
Ctrl + H 笔记内替换
F7 拼写检查
Ctrl + Enter 在新窗口打开
Ctrl + Alt + Enter 演示
Ctrl + Shift + Enter 在另一个屏幕演示
Ctrl + Alt + T 标签
Ctrl + Shift + E 电子邮件发送笔记
Ctrl + D 字体
Ctrl + L 左对齐
Ctrl + E 居中
Ctrl + R 右对齐
Ctrl + J 两端对齐
Ctrl + M 增加缩进
Ctrl + Shift + M 减少缩进
Ctrl + + 增大字体
Ctrl + - 缩小字体
Ctrl + Shift + B 项目符号列表
Ctrl + Shift + O 编号列表
Ctrl + B 粗体
Ctrl + I 斜体
Ctrl + U 下划线
Ctrl + T 删除线
Ctrl + Shift + H 高亮显示
Ctrl + Shift + - 插入水平标尺
Ctrl + Shift + C 插入复选框
Ctrl + K 添加超级连接
Ctrl + Shift + K 编辑超链接
Ctrl + Shift + R 删除超级链接
Ctrl + Shift + X 加密所选文本
Ctrl + Space 简化格式
Ctrl + Shift + Space 清除格式
Alt + Shift + D 插入日期和时间

笔记列表

以下快捷键将在笔记列表面板选中时起效。

快捷键 操作
↑ 前往上一条笔记
↓ 前往下一条笔记
Page Up 将笔记列表向上滚动一页
Page Down 将笔记列表向下滚动一页
Home 前往列表中的第一条笔记
End 前往列表中的最后一条笔记
Delete 移到废纸篓
Ctrl + Enter 在单独的窗口中打开选定的笔记(ALT + F4关闭窗口)
Enter 将鼠标放在所选笔记的笔记编辑器中

左侧边栏

以下快捷键将在左侧边栏（笔记本列表、标签列表等）选中时起效。

快捷键 操作
↑ 将鼠标移动到该部分的上一个任务（笔记本、标签…）
↓ 将鼠标移动到该部分的下一个任务（笔记本、标签…）
→ 展开选中项目的子任务（如适用）
← 折叠选中项目的字任务（如适用）
Delete 删除所选项目
