icons8 针对Apple Google与微软风格设计的大量扁平图标下载

之前我在電腦玩物介紹過不少 ICON 圖示下載資源，你可以把這些 icon 圖示用在網頁設計、報告裝飾、簡報素材等需求當中，也因為這樣，不時會有朋友來信跟我推薦其他的免費圖示下載資源，今天要介紹的「 Icons8 」就是其中比較有特色的選擇之一。

「 Icons8 」最大的特點就是根據 Apple、 Google、 微軟等系統最新的扁平設計風格與指導原則去設計其 ICON 圖示，每一個圖案都會有 iOS、 Android 或 Windows 與 Office 專用的樣式變化可以分別下載。

而且「 Icons8 」上面數萬個 ICON 圖示都是由「 Icons8 」全新設計，帶來的好處就是所有圖示的風格統一，這讓你在使用上可以獲得一致的設計感。

對於需要使用一系列風格統一、設計精美圖示的朋友來說，在我之前介紹過的各個圖示下載服務之外，「 Icons8 」也是一個不錯的選擇：

  • icon 圖示免費下載！介面設計師用得到的9個免費圖庫
  • 完全免費 90 萬圖示下載搜尋引擎 Icon Ninja 最大免費圖示圖庫
  • 24 萬個專業 ICON 圖示免費！支援可商用向量圖下載

 

打開「 Icons8 」網站，你可以搜尋找到需要的圖示。

目前「 Icons8 」上有三萬多種圖示，包含各種系統的設計風格，以及每個圖示都有不同的 PNG 或向量圖檔尺寸，不過小尺寸圖檔提供免費下載，而大尺寸或向量圖檔則須付費下載。

你可以在左方分類的「 Very Basic 」中找到全部都免費下載的向量圖檔，這些圖檔大多是各個系統的基本功能按鈕，可以幫你快速使用在自己的網站、軟體介面上。

準備下載 Icon 圖示時，右方會看到 iOS、 Windows、 Android、 Office 等不同系統的切換按鈕，讓你可以針對圖案，選擇不同系統中的設計風格。

在「 Icons8 」上大多數圖示都有提供「小尺寸 PNG 」圖檔免費下載的選項。

遇到要付費下載的圖示，只要把尺寸調製到 PNG 的小尺寸，就能免費下載這個圖案。

另外「 Icons8 」還提供了 ICON 圖示線上客製化的功能，當你找到需要的圖案後，可以組合不同的圖標、加上不同外框，或是更改圖案顏色，調整成自己需要的風格後再下載。

「 Icons8 」上大多扁平化設計的 ICON 圖示，也都能讓我們自由更換顏色。

最後，「 Icons8 」還幾乎每天都推出新的圖示設計。

如果你要免費下載「 Icons8 」上的圖示，那麼記得使用時要在使用 ICON 的網頁中，或是產品說明頁中，註明圖示來源連結即可。

有需要統一風格設計，並且符合 Apple、 Google、 微軟扁平化設計風格圖示的朋友，可以到「 Icons8 」找找看有沒有你需要的圖標。
