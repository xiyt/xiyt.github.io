不使用任何插件修改Google Chrome的Usage Agent

作为软件开发从业者，或是前端测试人员，对浏览器的Usage Agent的应该不会陌生。它的表面含义是“使用代理”或“用户代理”，它是浏览器和网站服务器之前的一个桥梁，以便告知浏览器想以什么样的身份访问网站，网站服务器也会根据不同的Usage Agent，呈现给访问者不同的网站版本 ，例如网站界面和内容的差异。这就是问什么我们通过电脑和手机打开同一个网站，看到的风格是不一样的，因为这些网站根据访问者的Usage Agent进行不同的适配。

但是，有时候我们希望网页能够以我们指定的方式呈现，例如由于测试需要，我需要在电脑端浏览器展示网站的手机版风格，或者是在手机端，有些功能无法正常使用，需要手机浏览器展示PC版的内容。这时候就需要手动修改浏览器的Usage Agent。而Google Chrome内置提供了这个功能。今天就分享一下具体设置的方法。

1、Ctrl + Shift + I打开开发者工具，点击右上的三点式菜单，选择更多工具里的Network conditions选择

2、打开Network conditions后，去掉Usage agent里的自动选择框

3、选择Chrome里预设的Usage Agent，非常丰富，包括Chrome、IE、Firefox、Edge、Opera等浏览器，以及 Android和iPhone以及iPad等设备

4、自定义Usage Agent

如果以上还是没有你想要的Usage Agent，这里也支持自定义，选择Custom后，在下方的输入框里输入Usage Agent就可了。
