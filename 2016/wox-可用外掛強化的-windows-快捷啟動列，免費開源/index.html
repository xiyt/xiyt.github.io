[電腦玩物]Wox 可用外掛強化的 Windows 快捷啟動列，免費開源

前天介紹了「不只是優秀截圖軟體！ Snipaste 用貼圖創造多工神器」後，獲得很多回響，我自己也覺得有段時間沒有研究一下 Windows 上的高效率工具了，於是打開我筆記本裡長長的待辦研究清單，看看有沒有什麼還沒試過的軟體可以翻出來研究。

於是發現了這個叫做「 Wox 」的 Windows 快捷列軟體，她號稱是 Windows 上的 Alfred ，透過一條小小的搜尋列，可以開啟軟體、搜尋檔案、查詢網頁，並透過安裝外掛還能當作計算機、翻譯器、剪貼簿等來使用。

我實際測試後，覺得倒是可以用來取代久未更新的「 Launchy 」，除了功能相近且都可以透過外掛強化外，「 Wox 」是一個開源免費軟體，目前都還保持更新，而且在 Windows 上實際運作的速度很快。

啟動「 Wox 」繁體中文版

你可以到這個網頁下載「 Wox 」的最新軟體版本，找到「 Downloads 」處的清單，先選擇只下載「 Wox-最新版本號.exe 」這個安裝檔案即可。

下載後直接點擊安裝。

安裝完成，預設用［ Alt ］+［空白鍵］可以呼叫出「 Wox 」，這時候在快捷列輸入「 Settings 」，就能打開軟體設定，並且在語言處切換成繁體中文版。

 

在「 Wox 」快速搜尋軟體

呼叫出「 Wox 」後，可以輸入已安裝軟體的部分名稱，快速啟動常用的應用程式。

或者輸入 Windows 內建的各種功能、設定與控制台名稱，也能快速呼叫相應功能。

 

在「 Wox 」搜尋文件：安裝 Everything

如果你同時安裝知名的檔案名稱蒐尋軟體 Everything ，那麼就可以在「 Wox 」中利用關鍵字快速搜尋檔案與資料夾。

在「 Wox 」使用計算機

「 Wox 」內建了計算機功能，只要直接輸入算式，就能自動計算出結果。

在「 Wox 」啟動 Google 搜尋

「 Wox 」也內建了各種網路搜尋引擎，例如 Google 或 YouTube 等，以 Google 為例，只要在「 Wox 」輸入「 g 」，然後輸入要搜尋的關鍵字，就可以顯示搜尋建議，並打開 Google 搜尋。

如何擴充「 Wox 」外掛？

進入「 Wox 」設定的「外掛」頁面，可以利用［瀏覽更多外掛］連結來安裝額外功能。

安裝方式很簡單，進入特定外掛的頁面，會看到一列安裝指令，例如「 wpm install Clipboard History 」，複製下來，貼入「 Wox 」快捷列按下確認，就會自動進行安裝了。

剪貼簿外掛

例如我推薦可以試試看這個剪貼簿外掛，安裝指令是「 wpm install Clipboard History 」。

安裝完成後，只要在「 Wox 」輸入「 cb 」，就會看到最近複製過的內容，可以快速貼上。

雖然這樣的剪貼簿沒有「 Ditto 」那麼強大，但也實用。

翻譯外掛

或者你可以安裝有道翻譯的外掛，指令是「 wpm install 有道翻译 」。

之後在「 Wox 」輸入「 yd 」，然後輸入英文詞彙，就會即時看到翻譯。

 

天氣外掛

而輸入「 wpm install Weather 」可以安裝天氣外掛，讓我們隨時利用「 Wox 」查詢某城市最近五日的天氣。

更換「 Wox 」主題外觀

進入「 Wox 」設定的［主題］中，我們則可以替換不同的外觀。

如何移除「 Wox 」外掛？

最後，如果安裝了外掛後覺得用不到，也只要輸入「 wpm uninstall 」指令，然後選擇已經安裝的某一款外掛，便能移除。

相較之下，「 Wox 」不比 Mac 上要付費的 Alfred 那麼強大，但在 Windows 上則足以取代 Launchy 等工具，除了也有外掛功能外，「 Wox 」快速流暢的速度是我最喜歡它的一點。
