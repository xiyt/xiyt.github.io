第一时间更新印象笔记：大幅提升操作效率

今天Windows版印象笔记更新到了6.1.2，更新说明如下：

印象笔记 for Windows 6.1.2 版更新说明

新功能:

快捷搜索 使用 Ctrl + Q，轻松实现在笔记本、标签及搜索之间快速切换 更快在私人及企业内容之间完成指定笔记本和标签的切换 笔记缩放：Ctrl+”+”, Ctrl+”－”, Ctrl+”0” 性能提升:

标签视图回来啦！随时用它来筛选、编辑和整理你的标签吧 直接在笔记编辑器中显示并编辑笔记日期 拍照并直接插入笔记中

其中带来了三个主要的更新项目，我们来一同体验一下：

1、快捷搜索面板Ctrl + Q

印象笔记新增这一功能主要用来搜索笔记本、标签和已保存的搜索，通过搜索结果快速定位的相应的笔记本、标签和已保存的搜索，这个比在左侧的导航窗口中一点一点寻找方便多了。不过，我倒是觉得如果能把这个Ctrl + Q快捷搜索完善成对印象笔记里的所有内容包括笔记内容的搜索将更加方便。不容把鼠标定位到笔记搜索框，一个Ctrl+Q，打开全局搜索。这只是个建议。

     

2、笔记窗口的缩放功能

现在在笔记窗口可以通过以下快捷键来缩放笔记了：

  • Ctrl + “+”：放大
  • Ctrl + “-“：缩小
  • Ctrl + “0”：还原

这个功能在一定程度上能够替代笔记演示功能，如果你不是印象笔记的高级会员，那么笔记演示功能是无法使用的。但是有了这个笔记放大的功能，一定程度可以替代笔记演示。

3、全局的标签视图

在早期版本中是有标签视图的，后取消掉了。这次的版本升级后，又将这一有用功能恢复回来了，并且更好易用了。整个标签视图按照字母顺序排列，也可以通过上方的搜索框对标签进行搜索。双击标签就可以显示按照这个标签搜索的笔记结果。
