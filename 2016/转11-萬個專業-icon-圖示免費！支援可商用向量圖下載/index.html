[转]11 萬個專業 ICON 圖示免費！支援可商用向量圖下載

在連續假期中，昨天先寫了一篇比較嚴肅的時間管理文章：「雲端時代時間管理術(上)：如何當擁有更多時間的人」，今天早晨想說應該先切換一下心情，大家都在放假，應該看一點輕鬆的文章，那麼，就來介紹一個之前沒有好好深入推薦的免費資源網站吧！

文章開頭的影片裡，我搭配旁白介紹了「 FLATICON 」這個號稱是最大的「免費，可商業使用」圖示素材下載網站，以及其最重要的幾個特色。

他目前提供了十一萬個以上的 ICON 向量圖檔，並且每個月都還持續新增素材。

我們不需帳號就能下載這些向量圖庫，自由調整圖示的大小與顏色，並且使用在自己的網頁、海報、印刷品上，也授權使用在商業用途上，如果是免費授權，那麼我們只要在產品的說明處加上素材創作者來源的說明文字即可。 我之前在另外兩篇整理性的文章提過「 FLATICON 」：

  • icon 圖示免費下載！介面設計師用得到的9個免費圖庫
  • 向量圖庫免費下載！設計師一定要知道的五個專業圖庫

而現在要特別為文介紹，是因為後來我自己長期使用的過程中，發現「 FLATICON 」這個網站對我最有幫助，如果在眼花撩亂的「免費」向量圖庫網站中要挑選一個優先嘗試的服務，那麼我會推薦先試試看「 FLATICON 」。

1. 免費下載 11 萬個以上 ICON 圖示，持續新增

「 FLATICON 」每個月新增 4000 個以上的新 ICON 圖示，從多年前開始累積到現在，已經有 11 萬個以上的圖示可以搜尋並下載。

2. 不需帳號登入！優質圖示打包下載

在「 FLATICON 」上下載 ICON 圖示，不需要登入帳號，所有找到的 ICON 圖都能免費下載。並且「 FLATICON 」也幫同風格、同主題、同設計者的 ICON 打包整理，我們可以下載網站上分類好的圖示壓縮包。

同樣不需要帳號登入，進入某個圖示包的頁面後，點擊右方的［ Download Pack ］按鈕，就能一次下載全部圖檔。

3. 各種檔案格式下載： SVG、 PSD、 PNG、EPS 等

而且「 FLATICON 」上的所有圖示都提供了 PNG、 SVG、 PSD 等不同檔案格式，也就是支援向量圖檔。

例如下圖就是打包下載後，解壓縮可以看到圖示的所有不同檔案格式都在資料夾中。

4. 可自由調整大小尺寸、可自由幫 ICON 重新著色

既然「 FLATICON 」支援下載向量圖檔，也就是說我可以自由縮放其大小而不會失真，也可以幫圖示重新上色，這些都在授權可免費使用的範圍。

5. 免帳號，但需要標示素材授權來源

想要免費使用的用戶，只要注意其授權方式說明，絕大多數的 ICON 授權都是：「免費，可商業使用於網頁或印刷品，只要加上創作來源姓名標示即可」。所謂的加上姓名標示，允許像是在設計好的網頁最下方或說明文字處，加上類似「 Designed by XXXX 」的字樣即可。

另外「 FLATICON 」也提供付費帳號，這時候使用這些圖示就不需要加上創造素材來源說明。

6. 支援 Photoshop 等外掛

「 FLATICON 」還提供了 Photoshop、 illustrator 上的外掛（這類外掛，就類似：「Google 七款 Nik Photoshop 修圖工具今天免費釋出！」），讓我們在軟體工作時也能直接取用這些 ICON 圖檔，同樣的免費版會有每個檔案使用 ICON 數量的限制。對於簡報設計、海報製作、網頁版型來說，「 FLATICON 」都是一個值得一用的免費、可商用、專業高質感 ICON 圖示的下載網站，推薦給大家。
